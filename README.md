# Ejercicios de Programación Web

Este repositorio contiene una serie de ejercicios prácticos de **HTML, CSS, JavaScript y Bootstrap**, diseñados para reforzar los fundamentos del desarrollo web. Cada ejercicio está implementado en un archivo independiente y puede abrirse directamente en el navegador.

## Contenido de los ejercicios

### Ejercicio 1: Hola Mundo  
Creación de una página básica en HTML5 con la estructura mínima y un mensaje de bienvenida. Se practica la organización de carpetas en XAMPP y el uso de etiquetas esenciales como `<h1>` y `<p>`.

### Ejercicio 2: Párrafos y Formato Básico  
Uso de etiquetas para dar formato al texto: `<p>`, `<strong>`, `<em>` y `<br>`. Se introduce la semántica básica para resaltar contenido.

### Ejercicio 3: Enlaces Simples  
Creación de hipervínculos externos e internos con la etiqueta `<a>`. Se utilizan atributos como `target="_blank"` y `target="_self"` para controlar la apertura de enlaces.

### Ejercicio 4: Imágenes Básicas  
Inserción de imágenes en una página web mediante `<img>`, con atributos `src`, `alt` y `width`. Se refuerza la importancia de la accesibilidad y la correcta organización de archivos.

### Ejercicio 5: Combinando Texto e Imágenes con Enlaces  
Integración de texto, imágenes y enlaces en un mismo documento. Se practica la inserción de imágenes dentro de párrafos y la creación de enlaces a perfiles externos.

### Ejercicio 6: Secciones Semánticas  
Uso de etiquetas semánticas como `<header>`, `<section>` y `<footer>` para estructurar mejor el contenido de una página web.

### Ejercicio 7: Listas Anidadas  
Creación de listas ordenadas y desordenadas, incluyendo listas anidadas. Se aplica en un ejemplo de receta de cocina para organizar ingredientes y pasos de preparación.

### Ejercicio 8: Horario de Clases  
Diseño de una tabla con `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<td>` y `<th>`. Se aplica CSS para mejorar la presentación, incluyendo estilos para filas alternas y celdas especiales.

### Ejercicio 9: (Contenido bloqueado)  
Este ejercicio está relacionado con prácticas de programación web, pero el documento no está disponible en este momento.

### Ejercicio 10: Conversión Celsius a Fahrenheit  
Implementación de una calculadora sencilla en HTML y JavaScript. El usuario ingresa grados Celsius y al presionar un botón se muestra el resultado en Fahrenheit en un campo de texto `readonly`. Incluye validaciones para asegurar que el valor sea numérico.

## Tecnologías utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5.3  
- Google Fonts  

## Cómo usar este repositorio
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/WilverAFS/Ejercicios_ProgramacionWeb.git
   ```
2. Abrir la carpeta en Visual Studio Code.  
3. Ejecutar cualquier archivo `.html` en el navegador para visualizar el ejercicio.
